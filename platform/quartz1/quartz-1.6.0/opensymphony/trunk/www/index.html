Project Info
OpenSymphony is an Open Source project dedicated to providing enterprise class J2EE applications and components. Our components aim to be specification compliant, and should work in any J2EE compliant application server. (Although some components only require a servlet container rather than a full J2EE app server).

Download and Installation
To download and install, go to the project page of the project you're interested in, and view the files available for download. Follow these simple steps:
  1. Download, install and start J2EE 1.4 SDK or any other J2EE compatible application server.
  2. Download the latest build of the project you're interested in from the Files Section.
  3. For applications that provide an example war or ear, deploy the example using your application server's deployment tool.

Build Instructions
Please review the build instructions for individuals projects, if available. Generally though, these steps should work:
  1. Download and install ant 1.5.1 or greater from http://ant.apache.org/, or use the command asant provided with the J2EE 1.4 SDK.
  2. Check out the project's source code from java.net. You can find instructions for doing so here in the 'version control' section of the project.
  3. Run 'ant'. A dist directory will be created with a jar file for the project.

Support
OpenSymphony is a community-supported open-source collection of projects. You can find support by subscribing to the various mailing lists offered in each individual sub-project..

Participation
If you are interested in contributing to any of the projects, whether it be with new features, suggestions, bugfixes, or documentation, then the best approach is to file issues and patches if applicable to the project's issue tracker
